# Playground

Monorepo dev setup:
- **Frontend**: Vite + React (workspace app: `apps/client`)
- **Backend**: FastAPI (Uvicorn, `apps/server`)

## Setup

```bash
pnpm install
```

## Run (frontend + backend)

```bash
pnpm dev
```

## URLs

- **Frontend**: `http://localhost:5173/`
- **Backend**: `http://127.0.0.1:8000/`
